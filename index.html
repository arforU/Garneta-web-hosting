Tentang Website Ini

Website ini bertujuan untuk memberikan edukasi seputar layanan web hosting kepada siswa, mahasiswa, maupun masyarakat umum. Diharapkan setelah membaca isi web ini, pembaca dapat memahami jenis-jenis hosting, prinsip kerja shared hosting, dan kelebihannya.

A. Pengertian Share Hosting Server

Layanan shared hosting sangat dipengaruhi oleh perilaku dari semua penyewa (klien) jasa hosting. Hal mendasar dalam memilih hosting biasanya adalah karena harga murah dan spesifikasi server yang bagus.

1. Shared Hosting

Shared hosting adalah layanan di mana sebuah server digunakan secara bersama-sama oleh banyak pengguna. Mereka berbagi resource & kapasitas server bersama, dengan pengaturan yang ditentukan oleh pengelola server. Kelemahannya, bila satu pengguna membuat masalah, pengguna lain juga bisa terkena dampaknya.

B. Prinsip Kerja Shared Hosting

Shared Hosting memungkinkan penggunaan satu server oleh banyak pengguna secara bersamaan. Semua website berjalan dalam sistem dan control panel yang sama. Pengguna tidak tahu siapa pengguna lain dan memiliki batasan penggunaan sumber daya.

C. Jenis-jenis Layanan Hosting

    Misalkan, menjalankan script robot atau script yang berjalan terus-menerus sehingga membebani server atau web/blognya menjadi lebih pelan, serta berbagai kemungkinan lain. Oleh sebab itu, penyedia hosting harus selalu memantau para penggunanya.
  1. Shared Hosting: Banyak pengguna berbagi satu server dengan hak akses terbatas.
  2. VPS Hosting: Server fisik dibagi menjadi server virtual yang berdiri sendiri.
  3. Cloud Hosting: Hosting dengan banyak server yang terhubung untuk menjaga performa situs tanpa downtime.
  4. Dedicated & Colocation Server: Dedicated menyewa dari datacenter, sedangkan Colocation membawa server sendiri untuk dikelola di datacenter.
  5. Wordpress Hosting: Dikhususkan untuk pengguna WordPress, lengkap dengan fitur tambahan seperti builder dan tema.
